About Text To MP3

What is this Web App?

Text To MP3 is a lightweight Flask-based web application that converts user-submitted text into high-quality speech (MP3 format). The app provides a simple web interface where users can input text, choose between male or female voice options, and download the generated MP3 file instantly.

Features

  • Convert text input into downloadable MP3 speech files
  • Support for both male and female voice options
  • Offline voice generation using pyttsx3 (no API calls required)
  • In-memory file streaming (no disk storage needed)
  • Dark mode friendly interface
  • Responsive design for desktop and mobile devices
  • Fast and efficient processing

How it works

Enter or paste your text into the text area on the home page, select your preferred voice (Male or Female), and click "Convert to MP3". The application uses the pyttsx3 library for text-to-speech synthesis, generating the audio file on-the-fly. Your MP3 file will be downloaded automatically, ready to use offline anytime!

Technology Stack

  • Backend: Flask (Python Web Framework)
  • TTS Engine: pyttsx3 (Text-to-Speech Library)
  • Frontend: Bootstrap 5, HTML5, CSS3
  • Icons: Bootstrap Icons
